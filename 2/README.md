# Дифференциальная диагностика ХАТ (болезнь Хашимото)

Веб-приложение для дифференциальной диагностики хронического аутоиммунного тиреоидита (гипертрофическая и атрофическая формы) на основе **программной интерпретации сети Петри**. Вся логика весов и переходов задаётся в `app/petri/hashimoto_net.json`.

## Требования

- Python 3.11+

## Установка и запуск

```bash
cd 2
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8001
```

Откройте http://127.0.0.1:8001

Порт **8001** выбран, чтобы не конфликтовать с проектом лимфомы Ходжкина (папка `1/`, порт 8000).

## Структура

| Путь | Назначение |
|------|------------|
| `app/petri/hashimoto_net.json` | Сеть Петри (единственный источник весов) |
| `app/petri/engine.py` | Движок: маркировка, переходы, финализация |
| `app/services/diagnosis.py` | Оркестрация опроса |
| `app/api/routes.py` | HTTP-маршруты и шаблоны |
| `logs/hashimoto.log` | Текстовый лог |
| `logs/hashimoto_audit.jsonl` | Audit по шагам |

## Тесты

```bash
pytest tests/ -v          # unit + сценарии + HTTP smoke (31 тест)
```

Smoke-тест (`tests/test_smoke_http.py`) проверяет, что все GET/POST маршруты опросника отвечают **200** (без редиректов).

## Мониторинг

`GET /health` → `{"status": "ok", "service": "hashimoto-diagnosis"}`

## Опросник

1. Жалобы (b1–b9)
2. Осмотр (b14–b20, пол и возраст — взаимоисключающий выбор)
3. Лаборатория (b23–b34, две ветки)
4. Инструментальные (b37–b40)

Результат: баллы `score_hyper` / `score_atrophic`, наиболее вероятная форма (b41 / b42). При 0/0 ошибки нет.
