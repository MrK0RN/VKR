# VKR — два независимых проекта

Репозиторий содержит два отдельных веб-приложения. Каждое живёт в своей папке, со своим Docker, зависимостями и документацией. Общего runtime между ними нет.

| Проект | Папка | Описание |
|--------|-------|----------|
| Лимфома Ходжкина | [`1/`](1/) | Дифференциальная диагностика нодулярной и смешанно-клеточной форм |
| ХАТ (Hashimoto) | [`2/`](2/) | Дифференциальная диагностика гипертрофической и атрофической форм |

## Запуск

Каждый проект запускается из своей директории:

```bash
cd 1   # или cd 2
./scripts/run.sh
```

Или через Docker:

```bash
cd 1   # или cd 2
docker compose up -d --build
```

Подробности — в README соответствующей папки. В каждом проекте документировано:

1. **Структура** — фронт/бэк, каталоги, библиотеки
2. **Запуск и деплой** — зависимости, команды, переменные окружения, требования к серверу
3. **Клиент ↔ сервер** — маршруты, форматы форм и ответов, сессии
4. **Алгоритм расчёта** — передача признаков, веса, маркировка сети Петri, финализация с примерами JSON
