# Дифференциальная диагностика лимфомы Ходжкина

Веб-приложение на FastAPI с интерпретацией сети Петри из `hodgkin_net.json` (без if-else по симптомам вне движка).

## Запуск

```bash
cd 1
./scripts/run.sh
```

Откройте **http://127.0.0.1:8000/** (шаг 1: http://127.0.0.1:8000/step/1)

> Ошибка **-102** в браузере = сервер не запущен (connection refused). Сначала выполните `./scripts/run.sh` и не закрывайте терминал.

Альтернатива вручную:

```bash
source .venv/bin/activate
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

## Тесты и смоук

```bash
pytest tests/ -v
# или
./scripts/smoke.sh
```

Смоук-тест (`tests/test_smoke.py`) проверяет полный проход опросника: все GET/POST маршруты возвращают **200** (без редиректов 303).

## Структура

- `app/petri/` — загрузка сети, движок маркировки и переходов
- `app/services/` — сессии и оркестрация опроса
- `app/web/` — шаблоны Jinja2 и статика
- `hodgkin_net.json` — исходное описание сети (копия в `app/petri/`)

## Поток опроса

1. Жалобы → 2. Осмотр → 3. Лаборатория → 4. Инструментальные → Результат

На переходах между разделами при отсутствии весов показывается предупреждение (без блокировки). Для инструментального блока переход к b42/b43 срабатывает при ≥1 отмеченном признаке b44–b51.
